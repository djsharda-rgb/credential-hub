# Report Generation Prompt

## Purpose

Use this prompt template to generate status reports from weekly notes.

---

## Prompt

```
Given the following weekly notes for the KYB Identity Wallet project, generate a concise status report with these sections:

1. **Executive Summary** — 2-3 sentence overview of progress.
2. **Key Accomplishments** — Bullet list of completed items.
3. **Active Work Streams** — What is currently in progress.
4. **Risks & Blockers** — Issues that need attention.
5. **Next Steps** — Planned work for the coming week.

Format the report in Markdown. Keep it under 500 words.

Weekly Notes:
[PASTE WEEK NOTES HERE]
```
