# Reports

This directory contains generated status reports for the KYB Identity Wallet project.

## How to Generate Reports

1. Update `inputs/week_notes.md` with the current week's progress.
2. Use the prompt template in `inputs/report_prompt.md` to generate a report.
3. Save the generated report in this directory with the naming convention: `YYYY-MM-DD-status-report.md`.
